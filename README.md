## Features

- Beautiful modern music player
- Play, Pause, Next & Previous controls
- Live progress bar
- Volume control
- Add & Remove Favorites
- Recently Played Songs
- Real-time Search
- Music Categories
- Playlists
- Glassmorphism UI
- Animated glowing background
- Custom glowing cursor
- Responsive Design

## Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/BeatWave.git
```

Go into the project

```bash
cd BeatWave
```

Install dependencies

```bash
npm install
```

Run the project

```bash
npm run dev
```
